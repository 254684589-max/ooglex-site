关于本页数据

本页汇集美国财政部下属「金融研究办公室」（Office of Financial Research，OFR）的五大监测工具：金融压力指数（每日，0 为历史平均压力，正值高于平均、负值低于平均）、短期融资监测（每日，SOFR/EFFR 等隔夜利率与成交量）、货币市场基金规模（每月）、对冲基金监测与银行系统性风险监测（每季度）。数据来源为 OFR 公开数据接口（金融压力指数 CSV、短期融资监测 STFM API 等），在服务端定时抓取后静态托管。所有数据仅供参考，不构成投资建议。
