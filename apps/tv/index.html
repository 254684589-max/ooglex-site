准备就绪

📲 安装到手机

把「环球TV」装到主屏，像 App 一样全屏打开，无需应用商店。

🤖 安卓（Chrome / Edge / 三星等浏览器）
  1. 点浏览器右上角菜单 ⋮
  2. 选择「安装应用」或「添加到主屏幕」
  3. 确认后，桌面会出现「环球TV」图标
🍎 iPhone / iPad（用 Safari 打开）
  1. 点底部的分享按钮 ⬆️
  2. 向下滑动，选择「添加到主屏幕」
  3. 点右上角「添加」，主屏即出现 App 图标
💬 微信 / QQ / 其它内置浏览器
  1. 内置浏览器可能无法播放视频
  2. 点右上角 ··· → 「在浏览器打开」（或用系统 Safari / Chrome 打开本页）
  3. 再按上面对应系统的步骤安装

📺 环球TV

华语频道直连优先 · 无需登录

转动地球，蓝点是一座座正在直播的电视台。点亮任意一个，就能实时观看当地的画面。

🎯 拖动地球，把想看的地方转进中间的圆圈，松手即自动观看
🔵 也可直接点击任意蓝点观看该台
🔍 顶部搜索城市 / 国家 / 频道名
🎲 随机一台 · 🏠 回到中国上空 · ↻ 自转

说明：仅收录 HTTPS 直连的公开直播源，华语区及多数亚洲频道可直接观看。连不上的台会自动跳到附近可用频道；部分欧美频道服务器在墙外，国内需代理才行，属正常现象。

📱 在 iPhone / iPad 上安装：点击 Safari 底部的分享按钮 ⬆️， 向下滑动选择「添加到主屏幕」，即可像 App 一样全屏打开。
