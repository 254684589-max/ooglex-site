📲 安装到手机

把「环球电波」装到主屏，像 App 一样全屏打开、离线也能启动，无需应用商店。

🤖 安卓（Chrome / Edge / 三星等浏览器）
  1. 点浏览器右上角菜单 ⋮
  2. 选择「安装应用」或「添加到主屏幕」
  3. 确认后，桌面会出现「环球电波」图标
或 ⬇️ 直接下载 APK 安装（无需商店）
🍎 iPhone / iPad（用 Safari 打开）
  1. 点底部的分享按钮 ⬆️
  2. 向下滑动，选择「添加到主屏幕」
  3. 点右上角「添加」，主屏即出现 App 图标
💬 微信 / QQ / 其它内置浏览器
  1. 内置浏览器无法直接安装
  2. 点右上角 ··· → 「在浏览器打开」（或用系统 Safari / Chrome 打开本页）
  3. 再按上面对应系统的步骤安装

🌍 环球电波

国内直连优先 · 无需登录

转动地球，绿点是一座座正在广播的电台。点亮任意一个，就能实时收听当地的声音。

🎯 拖动地球，把想听的地方转进中间的圆圈，松手即自动收听
🟢 也可直接点击任意绿点收听该台
🔍 顶部搜索城市 / 国家 / 电台名
🎲 随机一台 · 🏠 回到中国上空 · ↻ 自转

说明：仅收录 HTTPS 直连电台，国内及多数亚洲电台可直接收听。连不上的台会自动跳到附近可用电台；部分欧美电台服务器在墙外，国内需代理才行，属正常现象。

📱 在 iPhone / iPad 上安装：点击 Safari 底部的分享按钮 ⬆️， 向下滑动选择「添加到主屏幕」，即可像 App 一样全屏打开。
